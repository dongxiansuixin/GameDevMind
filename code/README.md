# 🎮 GameDevMind 配套代码示例

本目录包含《游戏开发技术图谱》的配套可运行代码示例，每个示例直接对应 `mds/` 中的知识文档。

## 📂 目录结构

```
code/
├── 1.基础能力/
│   ├── 1.1.2.C++语言/
│   │   ├── memory_pool/        ← 游戏服务器内存池
│   │   └── smart_pointer/      ← 智能指针最佳实践与陷阱
│   ├── 1.2.1.设计模式/
│   │   ├── command/            ← 命令模式（游戏输入/回放）
│   │   └── object_pool/        ← 对象池（子弹/粒子系统）
│   └── 1.2.2.数据结构/
│       └── quadtree/           ← 四叉树（空间分区/碰撞检测）
├── 2.技术能力/
│   └── 2.2.1.网络与通信/
│       └── network_sync/       ← 帧同步 vs 状态同步对比
└── 3.研发能力/
    └── 3.1.2.客户端3D场景开发/
        └── hex_grid/           ← 六边形网格 + A* 寻路
```

## 🚀 快速开始

每个示例目录包含独立的 `CMakeLists.txt` 或 `Makefile`，可以直接编译运行：

```bash
# 示例：编译并运行内存池示例
cd code/1.基础能力/1.1.2.C++语言/memory_pool
mkdir build && cd build
cmake .. && make
./memory_pool
```
